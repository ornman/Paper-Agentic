# 论文助手 - 流程架构图

## 1. PDF 导入与处理流程

```mermaid
flowchart TD
    A[用户上传 PDF] --> B[MinerU API 解析]
    B --> C[content_list_v2.json]
    B --> D[images/]
    C --> E[后端: 数据清洗层]
    D --> E
    E --> F[过滤噪声类型]
    F --> G[UI 噪声过滤]
    G --> H[LaTeX 清洗]
    H --> I[表格转换]
    I --> J[Chunk[] + 图片路径]
    J --> K[后端: VLM 图片描述]
    K --> L{Kimi Coding API}
    L -->|User-Agent: claude-code| M[10 并发请求]
    M --> N[图片详细描述]
    N --> O[Chunk[] + 描述]
    O --> P[后端: 混合切分策略]
    P --> Q{语义块 > 32k?}
    Q -->|否| R[保持原样]
    Q -->|是| S[平均切分]
    S --> T[首尾重叠 8k]
    T --> U[Chunk[] ~24k]
    R --> V[Embedding 服务]
    U --> V
    V --> W[硅基流动 Qwen3-Embedding-8B]
    W --> X[向量 4096 维]
    X --> Y[Qdrant 存储]
    Y --> Z[导入完成]
```

## 2. RAG 问答流程

```mermaid
flowchart TD
    A[用户提问] --> B[Query Rewrite Service]
    B --> C{检测查询类型}
    C --> D[LLM 改写优化]
    D --> E[改写后查询]
    E --> F[Retrieval Service]
    F --> G[Qdrant ANN 向量检索]
    F --> H[BM25 关键词检索]
    G --> I[RRF 融合]
    H --> I
    I --> J[Top-10 候选]
    J --> K[Rerank 服务]
    K --> L[硅基流动 Qwen3-Reranker-8B]
    L --> M[Top-5 精排]
    M --> N[QA Service]
    N --> O[构建 Prompt]
    O --> P[Kimi LLM 生成]
    P --> Q[流式输出 SSE]
    Q --> R[前端展示]
    R --> S[答案 + 来源 + 图片]
```

## 4. 混合切分策略

```mermaid
flowchart TD
    A[语义块] --> B{计算 Token 数}
    B --> C{<= 32k?}
    C -->|是| D[保持原样]
    C -->|否| E[计算切分数]
    E --> F[n = ceil / 24k]
    F --> G[每块大小 = total / n]
    G --> H[计算重叠 = 32k - 每块]
    H --> I[切分 + 首尾重叠]
    I --> J[输出 n 个 Chunk]
    D --> K[输出 1 个 Chunk]
```

## 5. 前后端交互

```mermaid
sequenceDiagram
    participant U as 用户
    participant F as 前端
    participant B as 后端 API
    participant S as Services
    participant P as Processing
    participant Q as Qdrant
    participant L as LLM API

    U->>F: 上传 PDF
    F->>B: POST /api/v1/library
    B->>S: ingest_service.ingest()
    S->>P: 调用 MinerU API
    P-->>S: JSON + images
    S->>P: cleaner.clean()
    P-->>S: Chunk[]
    S->>P: describer.describe()
    P->>L: Kimi VLM (claude-code)
    L-->>P: 图片描述
    P-->>S: Chunk[] + 描述
    S->>P: chunker.chunk()
    P-->>S: Chunk[] (切分后)
    S->>P: embedder.embed()
    P->>L: 硅基流动 Embedding
    L-->>P: vectors
    S->>Q: 存储 vectors + metadata
    Q-->>B: 完成
    B-->>F: 导入成功
    F-->>U: 显示结果

    U->>F: 输入问题
    F->>B: POST /api/v1/query/ask
    B->>S: qa_service.ask()
    S->>P: query_rewrite.rewrite()
    P->>L: Kimi LLM
    L-->>P: 改写后查询
    S->>Q: 检索 (ANN)
    Q-->>S: 候选 chunks
    S->>P: reranker.rerank()
    P->>L: 硅基流动 Rerank
    L-->>P: 精排结果
    S->>L: Kimi LLM 生成答案
    L-->>S: 流式响应
    S-->>B: SSE chunks
    B-->>F: SSE stream
    F-->>U: 实时显示
```
