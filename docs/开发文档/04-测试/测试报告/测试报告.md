## 已知问题

1. **BM25 删除**: 全量重建索引，论文量大时可能慢
2. **切块器**: 不感知表格结构，复杂表格可能被切断
3. **清洗器**: 英文论文区域检测较弱
4. **Redis**: 本地未部署，对话/轮询功能未实际测试
5. **Embedding**: 大 batch 偶发 `RemoteProtocolError`，已有重试机制
