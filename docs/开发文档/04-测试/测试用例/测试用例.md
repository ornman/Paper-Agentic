## 1. 单元测试

### 1.1 存储层 (tests/test_crud.py)

#### TC-CRUD-001: Zvec 插入与检索
- **前置**: 空 Zvec 实例
- **步骤**: 插入 10 chunks → 用第一个向量查询 top5
- **预期**: 返回 ≥1 条结果，top1 score > 0.9

#### TC-CRUD-002: Zvec 按 paper_id 过滤
- **前置**: 插入 paper_A (5 chunks) + paper_B (5 chunks)
- **步骤**: 查询 `paper_id = 'paper_A'`
- **预期**: 只返回 paper_A 的 chunks

#### TC-CRUD-003: Zvec 删除论文
- **前置**: 插入 5 chunks
- **步骤**: delete_paper → 查询全部
- **预期**: doc_count = 0

#### TC-CRUD-004: BM25 索引与检索
- **前置**: 空 BM25 实例
- **步骤**: add_documents(10 docs) → query("VR 虚拟现实")
- **预期**: 返回 ≥1 条结果

#### TC-CRUD-005: BM25 删除
- **前置**: 插入 5 docs
- **步骤**: delete_paper
- **预期**: doc_count = 0

#### TC-CRUD-006: SQLite 插入与查询
- **步骤**: INSERT → SELECT
- **预期**: 正确读写 title

#### TC-CRUD-007: SQLite 更新
- **步骤**: INSERT → UPDATE title → SELECT
- **预期**: 新 title 生效

#### TC-CRUD-008: SQLite 删除
- **步骤**: INSERT → DELETE → count
- **预期**: count = 0
