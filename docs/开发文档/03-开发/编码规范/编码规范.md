# 编码规范 - 论文助手 MVP

> 原则: 能跑就行，不追求完美

---

## 1. Python 规范

### 基础

- Python 3.13+，类型注解必写（函数签名）
- 异步优先：所有 I/O 操作用 `async/await`
- 包管理：`uv`，依赖声明在 `pyproject.toml`

### 命名

```python
# 模块/文件: snake_case
# 类: PascalCase
# 函数/变量: snake_case
# 常量: UPPER_SNAKE_CASE

class ZvecStore:
    async def query_by_paper(self, paper_id: str, vector: list[float], topk: int = 10) -> list[Doc]:
        ...

MAX_CONTEXT = 32000
```

### 错误处理

```python
# 外部服务调用必须有 try/except
async def call_embedding(self, texts: list[str]) -> list[list[float]]:
    try:
        result = await self.client.embed(texts)
    except httpx.TimeoutError:
        raise ExternalServiceError(code=3001, message="Embedding 服务超时")
    except httpx.HTTPStatusError as e:
        raise ExternalServiceError(code=3002, message=f"Embedding 服务错误: {e.response.status_code}")
```

### 不可变数据

```python
# 用 dataclass(frozen=True) 或 Pydantic model，不原地修改
from dataclasses import dataclass

@dataclass(frozen=True)
class Chunk:
    paper_id: str
    content: str
    chunk_index: int
    source_page: int
```

### 文件大小

- 目标 200-400 行，上限 800 行
- 超过就拆模块

## 2. 前端规范

### 基础

- Vue 3 + TypeScript + Composition API
- `<script setup>` 语法
- 包管理：`pnpm`

### 命名

```typescript
// 组件: PascalCase
// 文件: PascalCase.vue
// 变量/函数: camelCase
// 常量: UPPER_SNAKE_CASE
// CSS 类: kebab-case
```

### 风格

- 功能优先，不花时间在样式上
- 浅绿色淡雅风格，别整花里胡哨的
- 半透明/显化的效果用 CSS transition 就行

## 4. 配置管理

### 硬编码禁止项

以下内容必须从 .env 或 SQLite config 表读取，不能硬编码：

- API Key
- 模型名称
- API URL
- 检索参数（TopK、权重等）

### 配置加载

```python
from pydantic_settings import BaseSettings

class Settings(BaseSettings):
    # LLM
    llm_base_url: str
    llm_api_key: str
    llm_model: str

    # VLM (Kimi)
    kimi_api_key: str
    kimi_base_url: str = "https://api.kimi.com/coding/v1/messages"
    kimi_model: str = "K2.6-code-preview"

    # Embedding
    siliconflow_api_key: str
    embedding_model: str = "Qwen/Qwen3-Embedding-8B"
    embedding_dimensions: int = 1536

    # 存储
    zvec_data_dir: str = "./data/zvec_db"
    redis_url: str = "redis://localhost:6379/0"
    redis_ttl: int = 3600

    model_config = {"env_file": ".env"}
```

- 完善的 CI/CD
- 国际化
- 暗色主题
- 无障碍
- 性能极致优化
