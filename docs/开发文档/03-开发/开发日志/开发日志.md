### 踩过的坑

| 坑 | 原因 | 解决 |
|----|------|------|
| Zvec LOCK 文件 | `open()` 失败但目录已存在 | `shutil.rmtree()` 后 `create_and_open()` |
| Zvec filter 语法 | `==` 不被识别 | 改为单等号 `=` |
| MinerU file_urls 类型 | 以为是 dict，实际是 string | 去掉 `.get("url")` |
| Embedding 连接断开 | 大 batch 导致远端关闭 | 加 `RemoteProtocolError` 重试 |
| 清洗 end_noise 遗漏 | early break 导致只找到最近的致谢 | 扫描全部标记位 |
| 切块过度合并 | 136 chunks 合并成 2 个巨块 | 改为固定 1000 token |
