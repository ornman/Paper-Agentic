# UI 交互说明 - 论文助手 MVP

> 版本: v2.0
> 更新日期: 2026-04-18
> 原则: 功能优先，丑就丑

---

## 1. 整体布局

WPS 插件 TaskPane，只有一个界面：**对话框**。

```
┌──────────────────────────┐
│  📚 论文助手              │  ← 标题栏
│                          │
│  [💬 对话] [📁 文献库]    │  ← 两个 Tab
│──────────────────────────│
│                          │
│  (内容区，随 Tab 切换)     │
│                          │
│                          │
│──────────────────────────│
│  [输入框]        [发送]   │  ← 底部固定
└──────────────────────────┘
```

- 整体浅绿色淡雅风格
- TaskPane 默认半透明，鼠标移入显化
- 宽度固定（WPS 侧边栏标准宽度）

```
│                          │
│  🤖 Transformer在自然语言 │
│     处理领域应用广泛，主要 │
│     包括：                │
│     1. 文本分类...        │
│     2. 机器翻译...        │
│                          │
│     📎 来源:              │
│     • Attention Is All    │
│       You Need p.3 §3.2   │  ← 可点击，打开 PDF
│     • BERT p.5 §4        │
│                          │
│  👤 [圈选] cross-attention │
│     是怎么工作的？        │
│                          │
│  🤖 Cross-attention是一种 │
│     ...                  │
│                          │
│  [加载更多...]            │  ← 历史消息
└──────────────────────────┘
```

**消息类型**:

| 类型 | 显示 | 说明 |
|------|------|------|
| 用户文本 | 👤 灰底 | 用户输入的 prompt |
| 用户圈选 | 👤 灰底 + 🖊标记 | 圈选的文本高亮显示 |
| AI 回答 | 🤖 白底 | 流式输出，打字机效果 |
| 来源标注 | 📎 底部链接 | 可点击跳转本地 PDF |
| 系统消息 | ⚙️ 浅蓝底 | 导入完成、错误提示等 |

### 2.2 输入框交互

| 操作 | 效果 |
|------|------|
| 输入文字 + 点击发送 | 模式 4（有圈选时）或模式 1（纯提问） |
| 输入 `/start '路径'` + 发送 | 触发导入流程 |
| 圈选文字后点击发送 | 圈选内容自动填入 selection 字段 |
| 不输入任何内容，点击发送 | 模式 2（纯续写，基于轮询缓存） |

### 2.3 导入进度

导入过程中在对话区显示进度条：

```
┌──────────────────────────┐
│  📄 正在导入: xxx.pdf     │
│  ████████░░░░ 60%         │
│  当前步骤: 向量化中...     │
└──────────────────────────┘
```

导入完成自动显示系统消息，失败显示错误原因。

## 4. 轮询机制（前端）

### 4.1 后台轮询

```
WPS 文档内容 ──(5秒)──> 读取当前文档文本
                          ↓
                    POST /api/v1/poll/sync
                          ↓
                    后端缓存（Redis TTL 1h）
```

- TaskPane 加载后自动开始轮询
- WPS 关闭时停止轮询，缓存自动过期
- 重新打开时，首次轮询立即发送文档已有内容

### 4.2 圈选文字

用户在 WPS 中圈选文字后，前端通过 WPSJS API 获取选中文本，在发送时附带 `selection` 字段。
