## 技术栈

- **后端**: FastAPI (Python 3.13)
- **前端**: Vue 3 + TypeScript + Vite
- **向量库**: Qdrant
- **LLM**: Kimi Coding API
- **Embedding**: 硅基流动 Qwen3-Embedding-8B
- **部署**: WPS 插件
