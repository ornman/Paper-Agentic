### 会话表 (`sessions`)

**证据**: `models/session.py:11-19`

```python
class SessionORM(Base):
    __tablename__ = "sessions"

    id = Column(String, primary_key=True, default=lambda: str(uuid.uuid4()))
    title = Column(String, nullable=False)
    created_at = Column(DateTime, default=datetime.utcnow)
    updated_at = Column(DateTime, default=datetime.utcnow, onupdate=datetime.utcnow)

    # 关系
    messages = relationship("MessageORM", back_populates="session", cascade="all, delete-orphan")
```

**索引**:
- 主键: `id`
- 自动索引: `updated_at` (排序用)

## Qdrant Collection 配置

### Collection 命名规则

**证据**: `stores/qdrant_store.py:28-36`

```python
def _get_collection_name(paper_id: str) -> str:
    # Qdrant collection 名称限制：只能包含字母、数字、下划线、连字符
    name = f"paper_{paper_id}".replace("-", "_").replace(" ", "_")
    # 移除非法字符
    import re
    name = re.sub(r"[^a-zA-Z0-9_-]", "", name)
    return name.lower()
```

## 架构审查发现的问题

**￥问题 #3：失败导入残留文件系统数据￥**

**维度**: 架构与设计
**严重性**: P1
**位置**: `modules/ingestion/service.py:113-121`

**问题描述**:
PDF 导入失败时，文件系统中的 `data/papers/{paper_id}/` 目录不会自动清理，导致磁盘残留。

**代码证据**:
```python
# modules/ingestion/service.py:334-342
def _mark_failed(self, record: DocumentRecord, error: IngestionError) -> DocumentRecord:
    failed_record = record.transition("failed").model_copy(
        update={
            "error_stage": error.stage,
            "error_message": error.message,
        }
    )
    return self.repository.save_document(failed_record)
    # ❌ 没有清理文件系统
```

**潜在影响**:
- 磁盘空间泄漏
- 用户重试时可能使用旧数据
- 磁盘占用随失败次数增长

**建议方向**:
在 `_mark_failed` 中添加文件系统清理逻辑，或使用临时目录，成功后再移动到正式位置。

**￥问题 #4：BM25 索引未持久化到数据库￥**

**维度**: 架构与设计
**严重性**: P2
**位置**: `repositories/bm25_repo.py`（推断）

**问题描述**:
BM25 索引存储为 JSON 文件，而其他元数据都在 SQLite，数据管理不一致。

**代码证据**:
```python
# core/config.py:233-236
bm25_index_path: str = Field(
    default="./data/bm25_index.json",
    description="BM25 索引文件路径",
)
```

**潜在影响**:
- 数据一致性问题: BM25 索引与 SQLite 元数据不同步
- 备份困难: 需要分别备份 JSON 和 DB
- 并发问题: 文件锁不如数据库可靠

**建议方向**:
考虑将 BM25 索引入库 SQLite，或使用专门的键值存储（如 Redis）。
