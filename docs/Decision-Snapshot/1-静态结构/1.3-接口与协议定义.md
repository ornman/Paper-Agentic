## 安全边界

### 认证与授权

**当前状态**: ❌ 无认证

**证据**:
```python
# main.py:20-27
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 允许所有来源
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

**风险评估**:
- 🔴 **P0**: 任何人都可访问所有接口
- 🔴 **P0**: API Key 无保护，可通过 `/api/v1/config/` 获取（虽已脱敏，但仍可调用）
- 🔴 **P0**: 无速率限制，易被滥用

### 敏感信息脱敏

**配置脱敏策略**:
```python
# routes/config.py:37-41
def mask_key(key: str) -> str:
    """脱敏 API Key，保留前4位和后4位"""
    if not key or len(key) < 8:
        return "" if not key else "***"
    return key[:4] + "***" + key[-4:]
```

**证据**: `routes/config.py:37-41`

## 架构审查发现的问题

**￥问题 #5：无认证与授权机制￥**

**维度**: 安全与合规
**严重性**: P0
**位置**: `main.py:20-27`

**问题描述**:
所有 API 端点完全开放，无任何认证机制，任何人都可以调用。

**代码证据**:
```python
# main.py:20-27
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 允许所有来源
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# 所有路由都无 @auth 依赖
@router.post("/import", response_model=ApiResponse)
async def import_pdf(file: UploadFile):  # 无认证
    ...
```

**潜在影响**:
- 🔴 数据泄露: 会话记录、上传的 PDF 可被任何人访问
- 🔴 资源滥用: 恶意用户可无限制调用 API，消耗配额
- 🔴 API Key 泄露: 虽然配置接口脱敏，但仍可通过其他方式探测

**建议方向**:
1. 添加 API Key 或 JWT 认证
2. 添加速率限制
3. 限制 CORS 来源（当前为 `*`）

**￥问题 #6：敏感配置信息可被探测￥**

**维度**: 安全与合规
**严重性**: P1
**位置**: `routes/config.py:32-57`

**问题描述**:
虽然 API Key 已脱敏，但攻击者可通过测试接口探测服务可用性，进而发起攻击。

**代码证据**:
```python
# routes/config.py:60-88
@router.post("/test", response_model=ApiResponse)
async def test_config(request: ConfigTestRequest):
    ...
    result = await llm.chat([{"role": "user", "content": "hi"}])
    # 这会消耗真实的 API 配额
```

**潜在影响**:
- 🔴 配额浪费: 攻击者可反复调用测试接口消耗配额
- 🔴 信息泄露: 测试成功可推断 API Key 有效

**建议方向**:
1. 限制测试接口调用频率（如每分钟 1 次）
2. 记录测试调用日志
3. 考虑移除测试接口或添加认证
