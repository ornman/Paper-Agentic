```python
def run_pdf_task(self, file_path: str) -> dict:
    """提交PDF解析任务并轮询结果."""
    
    # 1. 提交任务
    task_id = self._submit_task(file_path)
    
    # 2. 轮询结果
    start_time = time.time()
    while True:
        if time.time() - start_time > self.timeout:
            raise IngestionError(...)
        
        result = self._get_result(task_id)
        if result["status"] == "completed":
            return result["payload"]
        
        time.sleep(self.poll_interval)
```

## 九、￥问题￥：依赖风险点

### 9.1 MinerU轮询延迟

**问题**：
- 5分钟超时，5秒轮询间隔
- 大文件可能长时间等待
- 占用线程资源

**影响**：
- 导入时间不可预测
- 可能触发客户端超时

**建议**：
- 前端显示进度
- 使用WebSocket推送状态

### 9.2 VLM并发限流

**问题**：
- 并发10可能触发API限流
- 无全局速率限制

**影响**：
- 高并发导入时可能失败
- 可能被API服务商封禁

**建议**：
- 添加全局速率限制
- 实现请求队列

### 9.3 外部依赖单点故障

**问题**：
- 所有依赖都通过HTTPS
- 代理故障导致全部不可用

**影响**：
- 系统完全不可用

**建议**：
- 添加本地缓存
- 实现降级模式
