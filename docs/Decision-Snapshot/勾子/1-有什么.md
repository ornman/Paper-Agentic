## 关键架构事实

1. **双架构并存**：`app/modules/`（新）和 `app/services/` + `app/services_archive/`（旧）。modules/ 按领域组织（ingestion/library/qa/retrieval/session），services_archive/ 已归档但仍留在代码树中。

2. **clients/ 与 modules/ingestion/ 下各有一份 mineru_client.py**：clients/ 下 5,313 字节（旧版），modules/ingestion/ 下 23,425 字节（新版，含自适应轮询），存在重复。

3. **LLMClient 未继承 ABC**：`app/clients/llm_client.py` 的 `LLMClient` 直接使用 openai SDK，未继承 `vlm_client.py` 中定义的 `LLMClient` 抽象类（命名冲突）。

4. **workflows/ 为空目录**：只有 `__init__.py`，无实际内容。

5. **前端无路由库**：无 vue-router，Tab 切换在 AppShell.vue 中用条件渲染实现。

6. **WPS 插件集成**：前端有两套 manifest/ribbon（根目录一套，wps-plugin/ 目录一套），构建产物在 dist/。
