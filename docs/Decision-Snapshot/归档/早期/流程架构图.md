## 2. 分布式 RAG 问答流程（核心）

```mermaid
flowchart TD
    A[用户提问] --> B[Query Rewrite Service]
    B --> C{检测查询类型}
    C --> D[LLM 改写优化]
    D --> E[改写后查询 1~3 个]

    E --> F["Retrieval Service:<br/>分布式并行检索"]

    F --> G["并行搜索所有 Collection"]
    G --> G1["paper_a_coll"]
    G --> G2["paper_b_coll"]
    G --> G3["video_xxx_coll"]
    G --> G4["note_xxx_coll"]

    G1 --> H["每 Collection Top-30"]
    G2 --> H
    G3 --> H
    G4 --> H

    H --> I["跨 Query RRF 融合"]
    I --> J["融合后 Top-30 候选"]

    J --> K[Rerank 服务]
    K --> L[硅基流动 Qwen3-Reranker-8B]
    L --> M[Top-10 精排]

    M --> N[QA Service]
    N --> O[构建 Prompt + 上下文]
    O --> P[DeepSeek/Kimi LLM 生成]
    P --> Q[流式输出 SSE]
    Q --> R[前端展示]
    R --> S[答案 + 来源 + 图片]
```

### RRF 融合详解

```
┌─────────────────────────────────────────────────────────┐
│  RRF (Reciprocal Rank Fusion) - 排名融合算法             │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  问题：不同 Collection 的分数可能不可比                  │
│  解决：只看排名，不看分数                                │
│                                                          │
│  公式：score[doc] += 1 / (k + rank + 1)                 │
│                                                          │
│  示例：                                                 │
│  ┌─────────────────┐     ┌─────────────────┐            │
│  │ Collection A    │     │ Collection B    │            │
│  ├─────────────────┤     ├─────────────────┤            │
│  │ doc1: 第1名     │     │ doc2: 第1名     │            │
│  │ doc2: 第2名     │     │ doc3: 第2名     │            │
│  │ doc3: 第3名     │     │ doc1: 第3名     │            │
│  └─────────────────┘     └─────────────────┘            │
│           │                       │                     │
│           └───────────┬───────────┘                     │
│                       ▼                                 │
│  ┌─────────────────────────────────────────┐            │
│  │ RRF 融合结果（k=60）                    │            │
│  ├─────────────────────────────────────────┤            │
│  │ doc2: 1/(60+2) + 1/(60+1) = 0.0325      │            │
│  │ doc1: 1/(60+1) + 1/(60+3) = 0.0323      │            │
│  │ doc3: 1/(60+3) = 0.0159                │            │
│  └─────────────────────────────────────────┘            │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

```python
# app/core/config.py
PINNED_EMBEDDING_MODEL = "Qwen/Qwen3-Embedding-8B"
PINNED_EMBEDDING_DIMENSIONS = 1536  # 固定
PINNED_RERANK_MODEL = "Qwen/Qwen3-Reranker-8B"
```
