### 4. **配置系统硬约束**（当前版本）
- **证据**：project/MVP/backend/app/core/config.py:286-298
- **关键约束**：
  ```python
  @model_validator(mode="after")
  def validate_pinned_model_contract(self) -> "Settings":
      """校验模型契约，防止向量维度和模型漂移污染索引。"""
      if self.embedding_model != PINNED_EMBEDDING_MODEL:
          raise ValueError(f"EMBEDDING_MODEL 必须固定为 {PINNED_EMBEDDING_MODEL}")
      if self.embedding_dimensions != PINNED_EMBEDDING_DIMENSIONS:
          raise ValueError(f"EMBEDDING_DIMENSIONS 必须固定为 {PINNED_EMBEDDING_DIMENSIONS}")
      if self.rerank_model != PINNED_RERANK_MODEL:
          raise ValueError(f"RERANK_MODEL 必须固定为 {PINNED_RERANK_MODEL}")
      return self
  ```
- **固定模型**：
  - Embedding: `Qwen/Qwen3-Embedding-8B`（1536 维）
  - Rerank: `Qwen/Qwen3-Reranker-8B`

## 核心架构变更（源码分析）

### 后端架构演进

#### 1. **导入链路完整化**（`IngestionService`）
- **证据**：project/MVP/backend/app/modules/ingestion/service.py:50-114
- **流程**：
  ```
  MinerU 解析 → 清洗 → VLM 图片描述 → 混合切分 → Embedding → Qdrant 存储
  ```
- **异步优化**：全链路异步（asyncio），避免阻塞 FastAPI 事件循环
- **错误处理**：统一 `IngestionError`，支持失败标记

#### 2. **混合检索系统**（`RetrievalService`）
- **证据**：project/MVP/backend/app/modules/retrieval/service.py:27-82
- **策略**：
  - 向量检索（召回 top_k * 2）
  - Rerank 二次排序
  - 预留 BM25 融合接口（未实现）
- **分布式支持**：每篇论文独立 collection

#### 3. **问答服务升级**（`QAService`）
- **证据**：project/MVP/backend/app/modules/qa/service.py:79-187
- **特性**：
  - Query 改写（多查询生成）
  - 流式输出（SSE）
  - RAG 上下文注入
  - 会话历史管理（SQLite）
