```python
PINNED_EMBEDDING_MODEL = "Qwen/Qwen3-Embedding-8B"

PINNED_RERANK_MODEL = "Qwen/Qwen3-Reranker-8B"
```

**校验逻辑**: `project/MVP/backend/app/core/config.py:286-298`

```python
@model_validator(mode="after")
def validate_pinned_model_contract(self) -> "Settings":
    """校验模型契约，防止向量维度和模型漂移污染索引。"""
    if self.embedding_model != PINNED_EMBEDDING_MODEL:
        raise ValueError(f"EMBEDDING_MODEL 必须固定为 {PINNED_EMBEDDING_MODEL}")

    if self.embedding_dimensions != PINNED_EMBEDDING_DIMENSIONS:
        raise ValueError(f"EMBEDDING_DIMENSIONS 必须固定为 {PINNED_EMBEDDING_DIMENSIONS}")

    if self.rerank_model != PINNED_RERANK_MODEL:
        raise ValueError(f"RERANK_MODEL 必须固定为 {PINNED_RERANK_MODEL}")

    return self
```

⚠️ **风险**: 一旦模型名或维度漂移，历史向量、检索结果和索引结构都会失去一致性

```typescript
 * 后端当前真实字段：
```

## 八、关键风险与缺失

### 8.1 🔴 高风险

1. **Embedding 模型固定契约** (`config.py:286-298`)
   - 一旦模型名或维度漂移，历史向量、检索结果和索引结构都会失去一致性
   - 需要索引迁移方案

2. **parent_child 索引模式已废弃** (`library/models.py:38`)
   - 仍在 `DOCUMENT_INDEX_MODES` 中，保留用于兼容性
   - 可能导致代码混乱

3. **旧版论文管理接口直接操作 Qdrant** (`library/routes/library.py:200-255`)
   - 与新架构的 `document_id` 可能不兼容
   - 需要统一迁移

4. **本地路径发送风险** (`api-client.ts:44-108`)
   - 导入接口会携带本地 PDF 路径
   - API base URL 被误配到外部主机时可能泄露本地文件结构

### 8.2 ⚠️ 中风险

1. **session 路由实现未在代码范围内体现**
   - 路由已注册但实现细节缺失
   - 需人工补充

2. **测试期望与实际路由不匹配**
   - `test_library_routes_are_registered` 期望的路由与 `library.py` 实现不一致
   - 可能导致测试失败或功能缺失

3. **SSE 事件字段名容错** (`sse-client.ts:197-217`)
   - 支持多种字段名（text, chunk, content, answer, delta）
   - 可能导致前后端字段名漂移

4. **来源卡片字段映射** (`sse-client.ts:239-273`)
   - 后端用 `document`/`content`，前端用 `title`/`snippet`
   - 需要显式映射，增加维护成本

### 8.3 ℹ️ 需人工补充

1. **session 路由具体实现**
   - 会话创建、查询、删除逻辑
   - 历史消息存储与检索

2. **配置热更新影响范围**
   - `reload_config()` 承认不影响已创建的单例对象
   - 需要明确哪些服务需要重启才能生效

3. **日志导出实现细节**
   - `export_logs_to_desktop()` 的具体实现未在代码范围内
   - 日志格式、导出范围需明确
