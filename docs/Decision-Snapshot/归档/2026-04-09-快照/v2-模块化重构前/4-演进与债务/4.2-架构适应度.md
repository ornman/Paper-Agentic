## 代码重复热点

| 重复模式 | 位置 | 说明 |
|---------|------|------|
| `httpx.AsyncClient(timeout=X)` | `kimi_client.py`, `embedding_client.py`, `rerank_client.py` | 超时值硬编码，每个客户端独立创建 |
| `get_settings()` 调用 | 几乎每个文件 | 配置单例获取重复，但 `@lru_cache` 保证效率 |
| `raise_for_status()` + 无 try/except | `kimi_client.py`, `embedding_client.py`, `rerank_client.py` | HTTP 错误处理模式相同但分散 |
| MinerU 客户端 | `clients/mineru_client.py` + `modules/ingestion/mineru_client.py` | **两份实现并存**（旧版+新版） |
| 旧版服务残留 | `services/cleaning_service.py`, `services/ingest_workflow.py`, `services/indexing_service.py` | 已迁移但未删除 |
