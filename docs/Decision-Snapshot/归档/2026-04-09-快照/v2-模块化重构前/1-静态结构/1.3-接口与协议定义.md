## 安全边界标注

| 端点 | 认证要求 | 敏感数据 | 风险 |
|------|---------|---------|------|
| 全部端点 | **无认证** | - | MVP 阶段无用户认证，任何人可访问 |
| `GET /config/` | 无 | API Key 脱敏（前4后4） | 脱敏逻辑存在（`config.py:36-39`） |
| `POST /library/import` | 无 | 上传文件路径 | 路径拼接，可能存在目录遍历风险 |
| `DELETE /library/papers/{id}` | 无 | 删除 Collection + 文件 | `shutil.rmtree` 无二次确认 |
| `POST /config/test` | 无 | 触发外部 API 调用 | 可被滥用消耗 API 配额 |
