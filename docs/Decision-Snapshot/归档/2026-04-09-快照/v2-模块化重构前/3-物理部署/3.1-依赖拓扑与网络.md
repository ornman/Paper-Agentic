## 依赖详细分析

### 1. DeepSeek API（主问答）

| 属性 | 值 |
|------|-----|
| 用途 | 主问答 LLM（qa_service.py、query_rewrite_service.py） |
| 协议 | HTTPS（OpenAI 兼容） |
| 地址 | `https://api.deepseek.com`（默认） |
| 认证 | `Authorization: Bearer {DEEPSEEK_API_KEY}` |
| 超时 | **未配置**（httpx 默认 5 秒） |
| 限流 | DeepSeek 官方限制 |
| 降级策略 | query_rewrite 失败 → 降级用原始查询 |
| 被谁调用 | `qa_service.py`、`query_rewrite_service.py` |

### 2. SiliconFlow API（Embedding + Rerank）

| 属性 | 值 |
|------|-----|
| 用途 | 文本向量化（Embedding）+ 重排序（Rerank） |
| 协议 | HTTPS（OpenAI 兼容） |
| 地址 | `https://api.siliconflow.cn/v1`（默认） |
| 认证 | `Authorization: Bearer {SILICONFLOW_API_KEY}` |
| 超时 | 60 秒（硬编码 `httpx.AsyncClient(timeout=60.0)`） |
| 批处理 | Embedding batch_size=32（可配置） |
| 降级策略 | **无** |
| 模型契约 | **锁定**（Qwen3-Embedding-8B + Qwen3-Reranker-8B） |
| 被谁调用 | `embedding_client.py`、`rerank_client.py` |

### 3. MinerU API（PDF 解析）

| 属性 | 值 |
|------|-----|
| 用途 | PDF → JSON + 图片（远程解析） |
| 协议 | HTTPS |
| 地址 | `https://mineru.net/api/v1`（默认） |
| 认证 | `Authorization: Bearer {MINERU_API_KEY}` |
| 超时 | 提交 120s / 状态查询 30s / 下载 120s / 整体 300s |
| 轮询 | `time.sleep(poll_interval)` 同步阻塞 |
| 降级策略 | **无**（唯一 PDF 解析路径） |
| 被谁调用 | `clients/mineru_client.py`、`modules/ingestion/mineru_client.py` |

### 4. Kimi Coding API（VLM + LLM）

| 属性 | 值 |
|------|-----|
| 用途 | VLM 图片描述 + RAG 问答流式输出 |
| 协议 | HTTPS（非标准 API） |
| 地址 | `https://api.kimi.com/coding/v1`（默认） |
| 认证 | `Authorization: Bearer {KIMI_API_KEY}` |
| 超时 | 120 秒（硬编码） |
| **特殊** | **必须伪装 `User-Agent: claude-code`** |
| 降级策略 | **无**（VLM 描述无替代方案） |
| 被谁调用 | `kimi_client.py` → `describer.py`、`qa_service_rag.py` |

### 5. Qdrant（向量库）

| 属性 | 值 |
|------|-----|
| 用途 | 向量存储 + 相似度检索 |
| 部署 | 本地文件（默认 `./data/qdrant`），可选远程 |
| 认证 | 远程模式需 API Key，本地无 |
| 降级策略 | **无**（检索完全依赖） |
| 被谁调用 | `qdrant_store.py` ← `retrieval_service.py`、`ingest_service.py` |

### 6. SQLite（元数据）

| 属性 | 值 |
|------|-----|
| 用途 | 会话、消息、元数据 |
| 部署 | 本地文件（默认 `./data/app.db`） |
| 认证 | 无 |
| 降级策略 | **无**（会话管理完全依赖） |
| 被谁调用 | `sqlite_repo.py` ← 路由层、服务层 |

## 不可用影响表

| 依赖断了 | 影响范围 | 功能失效 | 当前降级 |
|---------|---------|---------|---------|
| **DeepSeek API** | 问答、Query 改写 | 问答不可用，改写降级为原始查询 | Query 改写有降级 |
| **SiliconFlow Embedding** | 导入 + 检索 | 无法导入新论文，无法向量化查询 | **无** |
| **SiliconFlow Rerank** | 检索排序 | 检索质量下降（仍可用原始分数） | **隐式降级**：Rerank 失败可跳过 |
| **MinerU API** | PDF 导入 | 完全无法导入论文 | **无**（唯一解析路径） |
| **Kimi Coding API** | VLM 描述 + RAG 问答 | 图片无描述，RAG 问答不可用 | **无** |
| **Qdrant** | 检索 + 存储 | 检索不可用，无法入库 | **无** |
| **SQLite** | 会话管理 | 会话/消息不可用 | **无** |

## 依赖方向法则

> **箭头指向谁，就被谁绑架。**

```
论文助手后端 ──被绑架──→ DeepSeek API     (主问答)
             ──被绑架──→ SiliconFlow      (Embedding + Rerank)
             ──被绑架──→ MinerU API       (PDF 解析)
             ──被绑架──→ Kimi Coding API  (VLM + 流式问答)
             ──控制──→  Qdrant           (本地进程)
             ──控制──→  SQLite           (本地文件)
```

**4 个公网依赖全部无降级**，任一不可用将导致对应功能完全中断。
