# 3.3 环境差异矩阵

## 环境对比表

| 维度 | 本地开发环境 | WPS 插件环境 | Docker 容器 | 备注 |
|------|-------------|-------------|------------|------|
| **文件系统** | Windows NTFS | Windows NTFS | Linux ext4 | 路径分隔符差异 |
| **Python 版本** | 3.13+ | 3.13+（需打包） | 3.13+ | WPS 需要嵌入式 Python |
| **后端部署** | 直接运行 `python main.py` | 本地 HTTP 服务 | 容器内运行 | WPS 调用本地 localhost |
| **前端部署** | 独立浏览器 | WPS 内置 WebView | 无需前端 | WPS 提供容器 |
| **数据目录** | `./data/` | `%APPDATA%/论文助手/data/` | `/app/data/` | 需配置绝对路径 |
| **配置文件** | `.env` | `%APPDATA%/论文助手/.env` | 环境变量注入 | 路径需硬编码 |
| **日志输出** | 控制台 | 文件 + WPS 调试窗口 | stdout（容器日志） | 日志级别不同 |
| **依赖安装** | `uv sync` | 打包进插件 | Dockerfile 构建 | WPS 需要离线安装包 |

## WPS 环境特有问题

### ￥问题￥：路径处理差异

**【代码事实】Windows 路径分隔符

**文件位置**：`app/modules/library/service.py:162-167`

```python
# 使用 os.path 而非 Path 进行文件存在性检查
# 原因：Windows 文件系统对中文引号等特殊字符的处理更稳定
if not normalized_file_path.lower().endswith(".pdf"):
    raise ValueError("import_pdf 只接受 .pdf 文件")
if not os.path.isfile(normalized_file_path):
    raise ValueError(f"PDF 文件不存在: {normalized_file_path}")
```

**【模型推断】WPS 环境的差异**

| 场景 | 本地开发 | WPS 插件 | 后果 |
|------|---------|---------|------|
| 当前目录 | `project/MVP/backend/` | `C:\Program Files\WPS Office\` | 🔴 数据写入错误位置 |
| 相对路径 `./data` | `backend/data/` | `WPS Office/data/` | 🔴 权限错误 |
| 中文路径 | ✅ 支持 | ⚠️ 可能乱码 | 🔴 文件找不到 |
| 长路径 | ✅ 支持 | ⚠️ 260 字符限制 | 🔴 路径截断 |

### ￥问题￥：文件系统权限

**【模型推断】WPS 安装目录的权限限制

| 目录 | 权限 | 影响 |
|------|------|------|
| `C:\Program Files\WPS Office\` | 只读 | 🔴 无法写入数据 |
| `%APPDATA%/论文助手/` | 读写 | ✅ 应该使用 |
| `%TEMP%/` | 读写 | ⚠️ 重启后清空 |
| 用户文档目录 | 读写 | ✅ 可以使用 |

**建议**：使用 `APPDATA` 环境变量

```python
# 伪代码（未来扩展）
import os

        return Path(home) / ".论文助手"

# 配置
data_dir = get_data_dir()
```

### ￥问题￥：网络环境

**【模型推断】WPS 环境的网络差异

| 场景 | 本地开发 | WPS 插件 | 后果 |
|------|---------|---------|------|
| 代理设置 | Clash / 系统代理 | 可能无代理 | 🔴 API 调用失败 |
| 防火墙 | 开发环境开放 | 企业环境受限 | 🔴 端口被拦截 |

| 超时时间 | 默认 120 秒 | 企业网络更慢 | ⚠️ 需要更长超时 |

**建议**：添加可配置的超时和代理

```python

class ProxySettings:
    """代理配置."""

    http_proxy: str | None = Field(default=None)
    https_proxy: str | None = Field(default=None)

        return proxies
```

## 环境检测逻辑

### 【代码事实】当前无环境检测

搜索 `os.name` / `platform` / `sys.platform`：

- ❌ **无**环境检测逻辑
- ❌ **无**WPS 特殊处理
- ❌ **无**路径适配逻辑

### 建议：环境适配层

```python
# 伪代码（未来扩展）

class Environment:
    """环境检测与适配."""

    @staticmethod
    def is_wps() -> bool:
        """检测是否运行在 WPS 环境中."""
        # 检查进程名
        import psutil
        current_process = psutil.Process()
        return "wps" in current_process.name().lower()

    @staticmethod
    def get_data_dir() -> Path:
        """获取适合当前环境的数据目录."""
        if Environment.is_wps():
            import os
            appdata = os.environ.get("APPDATA", "~")
            return Path(appdata) / "论文助手"
        else:
            return Path("./data")

    @staticmethod
    def get_config_dir() -> Path:
        """获取配置文件目录."""
        if Environment.is_wps():
            import os
            appdata = os.environ.get("APPDATA", "~")
            return Path(appdata) / "论文助手"
        else:
            return Path(".")

    @staticmethod
    def setup_logging():
        """配置日志输出."""
        if Environment.is_wps():
            # 输出到文件
            log_file = Environment.get_data_dir() / "app.log"
            logging.basicConfig(
                filename=log_file,
                level=logging.INFO,
            )
        else:
            # 输出到控制台
            logging.basicConfig(
                stream=sys.stdout,
                level=logging.DEBUG,
            )
```

## 配置迁移策略

### 【代码事实】当前配置加载逻辑

**文件位置**：`app/core/config.py:278-284`

```python
model_config = SettingsConfigDict(
    env_file=".env",  # 🔴 硬编码
    env_file_encoding="utf-8",
    case_sensitive=False,
    populate_by_name=True,
    extra="ignore",
```

**问题**：
- 固定从当前目录加载 `.env`
- WPS 环境需要从 `%APPDATA%/论文助手/.env` 加载

### 建议：多环境配置支持

```python
# 伪代码（未来扩展）

def get_config_paths() -> list[Path]:
    """获取配置文件搜索路径."""
    paths = []

    # 1. 当前目录（开发环境）
    paths.append(Path(".env"))

    # 2. 用户配置目录（WPS 环境）
    if os.name == "nt":  # Windows
        appdata = os.environ.get("APPDATA", "~")
        paths.append(Path(appdata) / "论文助手" / ".env")

    # 3. 环境变量（Docker）
    # 不需要文件，直接从环境变量读取

    return paths

# 加载第一个存在的配置文件
config_path = next((p for p in get_config_paths() if p.exists()), None)
if config_path:
    settings = Settings(_env_file=config_path)
else:
    settings = Settings()  # 使用默认值
```
