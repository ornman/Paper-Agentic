## 模型固定契约

`config.py` 硬编码了三组不可变的模型契约，启动时 `model_validator` 强制校验：

```python
PINNED_EMBEDDING_MODEL = "Qwen/Qwen3-Embedding-8B"      # 1536 维
PINNED_EMBEDDING_DIMENSIONS = 1536
PINNED_RERANK_MODEL = "Qwen/Qwen3-Reranker-8B"
```

违反任何一个 → 服务启动失败。这是最严格的自保机制。

## 外部依赖契约

4 个外部 API，无任何 fallback：

| 服务商 | 用途 | API Key 来源 | 失效后果 |
|--------|------|-------------|----------|
| DeepSeek | 主 LLM 问答 | `.env` | QA 完全不可用 |
