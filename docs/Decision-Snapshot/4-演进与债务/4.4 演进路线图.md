**工作量**: 3天

**目标**:
- 导入流程命令化
- 支持灵活组合
- 易于扩展

**技术方案**:
```python
class IngestionCommand(ABC):
    @abstractmethod
    async def execute(self, context: IngestionContext) -> IngestionContext:
        ...

class IngestionPipeline:
    def __init__(self):
        self.commands = []
    
    def add_command(self, cmd: IngestionCommand):
        self.commands.append(cmd)
    
    async def execute(self, context: IngestionContext):
        for cmd in self.commands:
            context = await cmd.execute(context)
        return context
```

**验证**:
