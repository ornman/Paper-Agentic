### 5.2 缺失模式

**￥问题￥：缺少命令模式**

**问题**：
- 导入流程6个阶段串行执行
- 无法灵活组合/跳过阶段

**建议**：
```python
class IngestionCommand(ABC):
    @abstractmethod
    async def execute(self, context: IngestionContext) -> IngestionContext:
        ...

class ParseCommand(IngestionCommand):
    async def execute(self, context: IngestionContext) -> IngestionContext:
        ...

class CleanCommand(IngestionCommand):
    async def execute(self, context: IngestionContext) -> IngestionContext:
        ...

# 使用命令链
commands = [
    ParseCommand(),
    CleanCommand(),
    DescribeCommand(),  # 可选
    ChunkCommand(),
    EmbedCommand(),
    StoreCommand(),
]

for cmd in commands:
    context = await cmd.execute(context)
```

**￥问题￥：缺少观察者模式**

**问题**：
- 导入进度无法实时通知
- 前端无法感知状态变化

**建议**：
```python
class IngestionEventEmitter:
    def __init__(self):
        self.listeners = []
    
    def on_progress(self, callback):
        self.listeners.append(callback)
    
    async def emit_progress(self, stage: str, progress: float):
        for cb in self.listeners:
            await cb(stage, progress)
```
