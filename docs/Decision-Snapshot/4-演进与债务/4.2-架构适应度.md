## 代码重复热点分析

### HTTP 客户端重复模式

**位置**: `clients/` 目录下所有文件

**重复代码**:
```python
# kimi_client.py
async def _request(self, ...):
    async with httpx.AsyncClient(timeout=60.0) as client:  # 硬编码
        response = await client.post(...)
        response.raise_for_status()  # 无统一错误处理

# embedding_client.py
async def embed(self, texts: list[str]) -> list[list[float]]:
    async with httpx.AsyncClient(timeout=30.0) as client:  # 不同的超时值
        response = await client.post(...)
        response.raise_for_status()
```

**重复度**:
- 5 个客户端文件
- 每个文件约 30-50 行 HTTP 调用代码
- 总重复行数 ≈ 200 行

**建议**:
```python
# core/http_client.py - 统一 HTTP 客户端
class HttpClient:
    def __init__(self, timeout: float = 60.0):
        self.client = httpx.AsyncClient(timeout=timeout)
    
    async def post(self, url: str, **kwargs) -> httpx.Response:
        response = await self.client.post(url, **kwargs)
        response.raise_for_status()
        return response
```
