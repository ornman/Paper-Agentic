# 前端 UI 原型设计

> 创建日期：2026-03-23
> 状态：已确认
> 目的：UI 展示原型，不连接后端

## 1. 目标

创建一个纯前端 Vue 3 展示界面，用于演示 WPS 论文写作辅助工具的 UI 效果。

## 2. 设计方案

**方案 B：标签页切换布局**

- 顶部标签栏切换不同功能区
- 简单清晰，适合窄屏（WPS 侧边栏）
- 各功能独立展示，互不干扰

## 3. 界面布局

```
┌─────────────────────────────────────┐
│  [导入]  [对话]  [设置]             │  ← 标签栏
├─────────────────────────────────────┤
│                                     │
│         标签页内容区                 │
│                                     │
└─────────────────────────────────────┘
```

## 4. 组件结构

```
src/
├── App.vue
├── components/
│   ├── layout/
│   │   ├── TabBar.vue        # 顶部标签栏
│   │   └── TabContent.vue    # 标签页容器
│   ├── ingest/
│   │   ├── IngestPage.vue    # 导入页面
│   │   ├── PathInput.vue     # 路径输入
│   │   └── DocList.vue       # 文档列表
│   ├── chat/
│   │   ├── ChatPage.vue      # 对话页面
│   │   ├── SessionList.vue   # 会话选择
│   │   ├── MessageList.vue   # 消息列表
│   │   ├── MessageItem.vue   # 单条消息
│   │   └── ChatInput.vue     # 输入框
│   └── settings/
│       └── SettingsPage.vue  # 设置页面
├── stores/
│   └── mockStore.ts          # 模拟数据
└── assets/
    └── main.css
```

## 5. 三个标签页

### 5.1 导入页
- PDF 路径输入框 + 导入按钮
- 进度条（模拟动画）
- 已导入文档列表

### 5.2 对话页
- 会话选择下拉框
- 消息列表（用户 + AI + 来源卡片）
- 输入框 + 模式切换

### 5.3 设置页
- 服务地址（展示用）
- LLM 供应商选择
- 模型选择

## 6. 技术栈

- Vue 3 + TypeScript
- Vite
- Pinia（状态管理）
- 纯 CSS（现代 AI 助手风格）

## 7. 模拟交互

- 导入进度动画
- AI 回复逐字显示
- 来源卡片展示
- 会话切换
